🔌 ESP32 Connectors (P1 & P3/P4)

  • P1 Pin 1: GND
  • P1 Pin 2: RX (GPIO44) → ID Card TX
  • P1 Pin 3: TX (GPIO43) → ID Card RX
  • P1 Pin 4: +5V
  • P3/P4 SDA: GPIO19 → MCP I2C
  • P3/P4 SCL: GPIO20 → MCP I2C

📟 MCP23017 #1 (0x20)

  • A0-A7: IR Sensors 1-8 (Inputs)
  • B0-B5: Relays 1-6 (Outputs)
  • B6-B7: LED Indicators
  • VDD: +5V from ESP32
  • VSS: GND
  • A0/A1/A2: GND (Set address 0x20)

📟 MCP23017 #2 (0x21)

  • A0-A1: IR Sensors 9-10 (Inputs)
  • B0-B3: Relays 7-10 (Outputs)
  • B4: Water Pump Relay
  • B5: Door Lock Solenoid
  • A2-A7: Spare/Expansion
  • A0: VCC (address bit)
  • A1/A2: GND (address 0x21)

⚡ Power Supplies

  • 5V 5A: ESP32 + MCP23017 + Sensors + Relay Coils
  • 12V 10A: Relay COM → Motors + Water Pump
  • Common Ground: Connect all GNDs together
  • Relay COM: 12V+ from PSU
  • Relay NO: To Device (+)
  • Device (-): Direct to 12V GND
